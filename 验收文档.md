# SmartFarm 智慧农业系统验收文档

## 1. 项目名称

基于 LoRa 的智慧农业系统设计与实现

## 2. 项目简介

本项目以 STM32F103C8T6 + FreeRTOS 为核心，完成农场环境数据采集、本地显示、阈值报警、LoRa 无线传输、ESP32-S3 网关转发、MQTT 云端通信，以及 Web 和 Android 端可视化展示，形成一条完整的物联网闭环。

## 3. 系统目标

- 实现温湿度、土壤湿度、光照、雨滴等环境数据采集。
- 实现 OLED 本地显示和按键/旋钮交互。
- 实现异常报警和灌溉控制。
- 实现 STM32 到 ESP32-S3 的 LoRa 通信。
- 实现 MQTT 上云和多端展示。

## 4. 总体架构

系统链路如下：

```text
传感器/执行器 -> STM32F103 + FreeRTOS -> E32-433T20D LoRa -> ESP32-S3 网关 -> MQTT Broker -> Node-RED / Android
```

### 4.1 STM32 端

- 负责传感器采集、状态判断、阈值管理、蜂鸣器报警和 OLED 显示。
- 采用 FreeRTOS 任务拆分，提高实时性和可维护性。

### 4.2 ESP32-S3 端

- 接收 STM32 发送的 LoRa 数据。
- 连接 WiFi，并把数据发布到 MQTT 主题。

### 4.3 可视化端

- Node-RED 用于 Web 仪表盘展示。
- Android 客户端用于移动端查看实时数据与状态。

## 5. 主要功能

### 5.1 传感器采集

- AHT20 温湿度采集
- 土壤湿度采集
- 光照强度采集
- 雨滴检测

### 5.2 本地交互

- OLED 显示当前环境数据和阈值页面。
- KEY1 支持页面切换。
- KEY3 支持阈值浏览/编辑切换。
- 旋钮支持阈值项选择和数值调整。

### 5.3 报警与控制

- 环境参数超限时启动蜂鸣器报警。
- 支持继电器控制的自动灌溉逻辑。

### 5.4 无线与云端

- STM32 通过 USART3 连接 E32-433T20D 模块。
- 发送 JSON 格式数据到 ESP32-S3 网关。
- 网关将数据发布到 MQTT Broker。
- Web 和 Android 端订阅同一数据源展示结果。

## 6. 任务划分

系统采用 FreeRTOS 多任务结构：

- `SensorTask`：采集传感器并更新全局状态。
- `InputTask`：处理按键和旋钮输入。
- `ScreenTask`：刷新 OLED 显示。
- `LoRaTask`：周期发送农场数据到网关。

## 7. 通信格式

STM32 每秒发送一帧 JSON 数据，示例：

```json
{"temp":25.6,"hum":68.0,"soil":42,"light":520,"rain":0,"pump":1,"alarm":0}
```

字段含义：

- `temp`：温度
- `hum`：湿度
- `soil`：土壤湿度
- `light`：光照强度
- `rain`：雨滴状态
- `pump`：水泵状态
- `alarm`：报警状态
