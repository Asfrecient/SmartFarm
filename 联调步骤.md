# SmartFarm LoRa + MQTT 联调步骤

## 1. STM32 端

1. 在 CLion 中构建 `SmartFarm`。
2. 烧录到 STM32F103。
3. 确认 E32 接线：
   - PB10(TX3) -> E32 RXD
   - PB11(RX3) -> E32 TXD
   - 5V -> VCC
   - GND -> GND
   - M0/M1 -> GND

STM32 会每 1 秒通过 USART3 发送一帧 JSON。

## 2. ESP32-S3 网关

1. 用 Arduino IDE 打开：
   `esp32_s3/SmartFarmGateway`
2. 修改 `SmartFarmGateway.ino`：
   - `WIFI_SSID`
   - `WIFI_PASSWORD`
3. 安装 Arduino 库：
   - PubSubClient
   - ArduinoJson
4. 选择 ESP32-S3 板卡并上传。
5. 确认 E32 接线：
   - GPIO17(TX) -> E32 RXD
   - GPIO18(RX) -> E32 TXD
   - 3.3V -> VCC
   - GND -> GND
   - M0/M1 -> GND

ESP32 会把收到的 LoRa JSON 发布到 MQTT：

```text
broker.emqx.io:1883
topic: smartfarm/data
```

## 3. Android 端

1. Android Studio 打开：
   `android_smartfarm`
2. 运行 App。
3. 未收到 MQTT 时显示演示数据。
4. 收到 `smartfarm/data` 后自动切换在线模式。

## 4. Web 端

1. Node-RED 安装 dashboard 节点。
2. 导入：
   `web/smartfarm-node-red-flow.json`
3. Deploy 后打开 Dashboard。

## 5. 联通判断

如果链路正常：

1. STM32 OLED 显示传感器数据。
2. ESP32 串口无阻塞重启。
3. Android 首页数据从演示模式切到在线模式。
4. Node-RED Dashboard 仪表盘刷新。
