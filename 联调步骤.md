# SmartFarm LoRa + MQTT 联调步骤

## 1. STM32 端

1. 在 CLion 中构建 `SmartFarm`。
2. 烧录到 STM32F103。
3. 确认 E32 接线：
   - PB10(TX3) -> E32 RXD
   - PB11(RX3) -> E32 TXD
   - 5V -> VCC
   - GND -> GND
   - M0/M1 -> GND

STM32 会每 1 秒通过 USART3 发送一帧 JSON。
收到 LoRa 下发的控制 JSON 后，会切换水泵模式：

```json
{"pump":1}
{"pump":0}
{"pump":"auto"}
```

## 2. ESP32-S3 网关

1. 用 Arduino IDE 打开：
   `esp32_s3/SmartFarmGateway`
2. 修改 `SmartFarmGateway.ino`：
   - `WIFI_SSID`
   - `WIFI_PASSWORD`
3. 安装 Arduino 库：
   - PubSubClient
   - ArduinoJson
4. 选择 ESP32-S3 板卡并上传。
5. 确认 E32 接线：
   - GPIO17(TX) -> E32 RXD
   - GPIO18(RX) -> E32 TXD
   - 3.3V -> VCC
   - GND -> GND
   - M0/M1 -> GND

ESP32 会把收到的 LoRa JSON 发布到 MQTT：

```text
broker.emqx.io:1883
数据主题：smartfarm/data
控制主题：smartfarm/control
```

同时 ESP32 会订阅 `smartfarm/control`，收到 Android 发来的水泵控制命令后，通过 LoRa 转发给 STM32。

## 3. Android 端

1. Android Studio 打开：
   `android_smartfarm`
2. 运行 App。
3. 未收到 MQTT 时显示演示数据。
4. 收到 `smartfarm/data` 后自动切换在线模式。
5. 首页“水泵远程控制”可手动开启/关闭水泵，也可恢复自动控制。

## 4. Web 端

1. Node-RED 安装 dashboard 节点。
2. 导入：
   `web/smartfarm-node-red-flow.json`
3. Deploy 后打开 Dashboard。

## 5. 联通判断

如果链路正常：

1. STM32 OLED 显示传感器数据。
2. ESP32 串口无阻塞重启。
3. Android 首页数据从演示模式切到在线模式。
4. Android 点击水泵开关后，ESP32 串口打印控制命令，STM32 水泵状态随下一帧数据回传更新。
5. Node-RED Dashboard 仪表盘刷新。
